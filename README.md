# RuralLearn System

## 📘 Overview

RuralLearn System is an offline digital learning solution designed for rural secondary schools with limited or no internet access. It provides a local network-based platform where students and teachers can access educational resources such as digital textbooks, videos, and interactive learning tools.

## 🎯 Objectives

* Provide offline access to digital learning materials
* Improve ICT integration in rural schools
* Reduce dependency on internet connectivity
* Enhance student digital literacy

## 🏫 Case Study

This system is designed for **GS Kareba**, a rural secondary school facing challenges with internet access and ICT utilization.

## ⚙️ System Architecture

The system consists of:

* Local server (Linux-based)
* Network switch
* Client computers
* Wireless access points
* Offline LMS (Kolibri/Moodle)

## 🔌 How It Works

1. Users connect to the local network (LAN/WiFi)
2. Access the server via browser
3. Use offline LMS and digital resources
4. No internet required

## 🛠️ Technologies Used

* Linux Server
* Kolibri / Moodle (Offline LMS)
* Networking (LAN, Ethernet, WiFi)
* HTML/CSS (for interface if needed)

## 🚀 Installation Guide

1. Set up server (Linux)
2. Install Kolibri or Moodle
3. Configure LAN network
4. Upload educational content
5. Connect client devices
